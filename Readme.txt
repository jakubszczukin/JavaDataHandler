Program został napisany w języku Java SDK 17, IDE IntelliJ IDEA Community Edition 2021.2.3
Zostaje przeprowadzona walidacja dla kolumny "CPU Cores". Program nie pozwala na wprowadzenie jakiejkolwiek wartości poza liczbową w pola tej kolumny.
Zostaje również sprawdzona, czy jest wpisana wartość ujemna, bądź zostaje puste pole.
Po wpisaniu zedytowanych wartości, należy potwierdzić edycję klawiszem "Enter" w celu walidacji i zapisania danych. 
Po poprawnym zatwierdzeniu, do plików .txt/.xml zostaną wyexportowane zmodyfikowane dane.
W celu poprawnego działania programu, należy stosować z pliki .txt/.xml z formatem identycznym co podany schemat.

Do 4 części zadania została wykorzystana baza danych "laptops" wykorzystując system zarządzania MariaDB.
Wyeksportowana baza została załączona wraz z plikami projektu.
